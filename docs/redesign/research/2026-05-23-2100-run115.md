### NEW: Cloudflare Sandboxes GA — replace `/container`?
Agents Week 2026 (Apr 13-17) shipped Cloudflare Sandboxes to GA: persistent isolated Linux
environments, shell + filesystem + background processes, snapshot-based session recovery, PTY,
secure credential injection via egress proxy, filesystem watching, persistent code interpreters
[infoq.com/news/2026/04/cloudflare-sandboxes-ga](https://www.infoq.com/news/2026/04/cloudflare-sandboxes-ga/)
[blog.cloudflare.com/agents-week-in-review](https://blog.cloudflare.com/agents-week-in-review/).
Performance: cold start 30s (clone+npm install), 2s restore-from-snapshot. Pricing: active-CPU
only, up to 15,000 concurrent lite instances. **Direct candidate to replace vibesdk's custom
`worker/services/sandbox` + `/container` tooling** — same single-platform stack, snapshot
recovery would also obviate part of ADR-010 manual degraded-mode banner. Out of scope for Jul 1
launch; queue as Q3 2026 architecture review (NEW ADR candidate).
