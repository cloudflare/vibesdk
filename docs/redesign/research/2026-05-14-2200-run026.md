## Emergent Wingman Pivot — Strategic Shift

**What happened (April 15, 2026):**
Emergent launched Wingman — a WhatsApp/Telegram-based autonomous agent for scheduling, social media, sales, research, and hiring automation. Background agent, NOT a vibe-coding tool.

**Why this matters for vibesdk:**
Emergent is DE-EMPHASIZING its vibe-coding platform (8M builders, 1.5M MAU remain, but strategic energy has shifted). Wingman targets knowledge workers, not "builders." This is one fewer aggressive competitor in the non-dev vibe-coding segment.

**Opportunity:** Emergent's existing vibe-coding user base (8M builders) is now under-served by the parent company. Emergent's Wingman does NOT generate full-stack applications. Users who want to "build something" will look elsewhere. **vibesdk's messaging-first feature (WhatsApp/Telegram inbound webhook, S7) positions it as the CODING TOOL that fits inside the same messaging workflow Emergent is abandoning.**
