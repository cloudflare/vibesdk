## Implications for vibesdk
- **Add parallel sub-agent execution.** Three majors (Replit, Cursor, Devin) ship this in last 60 days. SimpleCodeGeneratorAgent state machine needs PHASE_IMPLEMENTING_PARALLEL fork support. Durable Object isolation actually makes this easier than competitors.
- **Ship MCP integration catalog now.** Notion/Linear/Jira/Slack/HubSpot are the bar. /worker/agents/tools/customTools.ts already has tool registration — wire MCP adapter.
- **Role-based project sharing (Viewer/Editor/Co-owner).** Trivial vs platform value; Bolt+Replit+Lovable all ship it. AppService + websocketTypes already model multi-user state.
- **Security scan on publish.** Run Semgrep or dependency-check before deploy succeeds. Differentiator dropping fast — will be table-stakes in 30 days.
- **PR review UI overhaul.** Cursor's split-PR + inline threads + file-tree picker is the new bar for AI-IDE PR experience.
