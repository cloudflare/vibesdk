## 6. Macro Threats / Tailwinds for vibesdk
**Tailwinds:**
- Cloudflare Agents Week 2026 productized Sandboxes, AI Gateway routing 14+ providers, Workers binding for 3rd-party models ([blog](https://blog.cloudflare.com/agents-week-in-review/)). vibesdk's CF-native architecture is now an officially blessed pattern.
- Lovable security crisis opens a **"trustworthy vibe-coding"** positioning lane. Sandbox isolation + secrets-DO encryption (vibesdk already ships these) are genuine differentiators.
- Anthropic prompt caching cuts cached input 90%, batch 50% off ([pricing](https://platform.claude.com/docs/en/about-claude/pricing)). vibesdk's heavy multi-phase agent loops benefit disproportionately.

**Threats:**
- **Cloudflare workforce cut 20%** — expect support/account-management gaps. Strategic risk for Workers-bound startups.
- **OpenAI GPT-5.4 at $2.50/$15** undercuts Sonnet 4.6 input pricing — provider arbitrage pressure on AGENT_CONFIG defaults.
- **Cursor $2B-ARR enterprise rollup** is consuming budget that would otherwise go to platform-builders.
- **Meta+Manus blocked by China** signals regulatory fragmentation; cross-border agent platforms face new friction.
- "Graduation arc" implies vibe-coding platforms are perceived as **disposable scaffolding**. vibesdk needs an answer: either own the graduation (export-to-IDE) or fight to keep users (production deploy story).
